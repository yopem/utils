# @yopem/try-catch

try catch wrapper for async functions

## Installation

```sh
npm install @yopem/try-catch
# or
pnpm add @yopem/try-catch
# or
yarn add @yopem/try-catch
# or
bun add @yopem/try-catch
# or
deno install npm:@yopem/try-catch
```
